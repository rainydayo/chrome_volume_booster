# Volume Booster

A simple Chrome extension that boosts the **current tab’s audio** up to **600%**.  
Useful for very quiet videos or audio (e.g. Google Drive, online classes, meetings).

---

## Features
- Boost tab audio from **0–600%**
- Minimal UI (single slider)
- Works per tab (Enable / Disable)
- Real-time processing using Web Audio API
- Does **not** modify the original audio file

---

## Installation (Developer Mode)
1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `volume-booster/` folder

---

## Usage
1. Open a tab with audio
2. Click the **Volume Booster** icon
3. Enable it for the current tab
4. Adjust the slider (up to 600%)

---

## Notes
- High boost levels may cause distortion on low-quality audio
- Slight latency may occur depending on the system
- Audio returns to normal when disabled or the tab is closed
